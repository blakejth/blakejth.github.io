CSS3 Transformations

What is it?

CSS Transforms and transformations are used to change the shape and/or location of an element

The best resource that I found can be reviewed here.

Two examples below

.box:hover { -ms-transform: rotate(30deg); -webkit-transform: rotate(30deg); transform: rotate(30deg); } .circle:hover { -ms-transform:scale(1.5,1.5); -webkit-transform:scale(1.5,1.5); transform:scale(1.5,1.5); }
rotate
Scale

Real world example

The following is an example that many photo websites use to indicate when a mouse is hovering an image. We can use transition to make the transform smooth.

.hoverActive { transition: all 1s ease-in-out; } .hoverActive:hover { transform: scale(1.05); }
