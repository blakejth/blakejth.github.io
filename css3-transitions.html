CSS3 Transitions

What is CSS3 Transitions?

"CSS transitions allows you to change property values smoothly (from one value to another), over a given duration." Rather than having properties change instantly, they can be modified with transitions to give more appeal. An example is the way that a window collapses on your computer. My personal favorite is on the Mac as it makes a cool animation. This transition lets the user know that the window didn't just dissapear but it minimized to a location where the user can then click again to resume.

https://www.w3schools.com/css/ css3_transitions.asp

What does it look like (code)

Syntax transition: property duration timing-function delay|initial|inherit;

Example code transition: opacity 0.3s;
